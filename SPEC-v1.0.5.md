### 数据联动修复
- `renderHome()` 每次进入时强制从 IndexedDB 读取最新 `K.DAILY` 和 `K.SESSIONS`
- 练习页点击返回时，强制调用 `updateEntryCounts()` + `renderHome()`
- 不依赖浏览器 back 按钮

## 2. 打赏功能

### 入口
- 顶栏右侧 `💝` 按钮，点击弹出 Modal

### Modal 内容
```
┌─────────────────────────────────┐
│  💝 支持百日闯             [✕]  │
├─────────────────────────────────┤
│  选择头像（点击选择）            │
│  🐱 🐶 🐰 🐼 🦊 🦁 🐸 🐵         │
│                                 │
│  署名（选填）：[___________]     │
│                                 │
│  微信收款码                     │
│  [图片区]                       │
│                                 │
│  支付宝收款码                   │
│  [图片区]                       │
└─────────────────────────────────┘
```

### 数据存储
- 贡献者名单存 IndexedDB `K.DONORS`（数组）
- 设置页新增「贡献者名单」tab

### 图片路径
- `images/qr/wechat.png`
- `images/qr/alipay.png`

## 3. Bug 修复

### 正确率不联动
- 修法：练习结束返回时，先写 IndexedDB，再强制调用 `renderHome()` 重新渲染
- 不依赖 `popstate` 或浏览器 back
