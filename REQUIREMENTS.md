## 允许的位置

- ✅ 设置页「关于」区块内的打赏按钮 + 收款码弹窗

## 禁止的位置

## 代码清理规范

移除功能时必须同时删除：
- [ ] HTML 标签和 modal
- [ ] CSS 类定义（不得留僵尸类名）
- [ ] JS action handler case
- [ ] 相关 localStorage / state 字段
- [ ] 相关图片资源

## 版本规则

- 大版本：第一位 +1（不兼容变更）
- 小版本：第二位 +1（功能增删）
- debug：第三位 +1（bug修复）
